

## Øk synligheten av bakgrunnsbildet

Bildet trenger litt mer gjennomslagskraft. Justerer opacity opp på bildet og ned på overlays:

| Element | Nåværende (landing / form) | Nytt |
|---|---|---|
| Bilde opacity | 0.50 / 0.35 | **0.60 / 0.45** |
| Cobalt overlay | 0.65 / 0.75 | **0.55 / 0.65** |
| Gradient from | 0.85 | **0.75** |
